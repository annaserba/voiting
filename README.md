# Генератор бюллетеня для голосования МКД

Проект создан на `Vue 3`, `Vite`, `TypeScript` и рассчитан на работу через `bun`.

## Что внутри

- форма для заполнения данных по дому, собственнику и управляющей компании;
- готовые вопросы голосования по:
  - реализации ранее принятого решения по шлагбаумам;
  - установке камер в подъездах и лифтах;
  - сбору средств через управляющую компанию;
  - созданию специального счета на ремонт шлагбаумов;
  - созданию счета для оплаты электроэнергии;
  - созданию счета для электроэнергии домофона на мусорной площадке;
  - сбору средств на систему открытия ворот;
  - установке домофона на калитки во внутренний двор от `Севдомофон`;
- печатная версия бюллетеня.
- разбиение на компоненты `ControlPanel`, `BulletinSheet` и `BulletinQuestion`.
- типизация данных бюллетеня через `TypeScript`.

## Запуск

```bash
bun install
bun run dev
```

## Сборка

```bash
bun run build
```

## Проверка типов

```bash
bun run check
```
